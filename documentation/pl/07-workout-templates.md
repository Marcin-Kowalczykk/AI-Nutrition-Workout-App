# Szablony treningowe

## Cel segmentu

Szablony pozwalają przyspieszyć tworzenie nowych treningów przez zapisanie powtarzalnej struktury.

## Główne ścieżki

- `/workout/template`
- `/workout/template/create`
- `/workout/template/[id]/edit`

## Funkcje

- tworzenie szablonu
- lista szablonów
- wyszukiwanie
- podgląd szablonu
- edycja
- usuwanie
- użycie szablonu jako bazy do nowego treningu

## Backend i API

- `app/api/workout-templates/create/route.ts`
- `app/api/workout-templates/get/route.ts`
- `app/api/workout-templates/list/route.ts`
- `app/api/workout-templates/update/route.ts`
- `app/api/workout-templates/delete/route.ts`
