# Exercises and Categories

## Segment goal

This segment manages the exercise dictionary that stabilizes naming across the application.

## Main route

- `/exercises`

## Functional scope

- create exercise categories
- create exercises
- list categories and exercises
- search
- delete categories
- delete exercises
- bulk delete mode

## Backend and API

- `app/api/exercises/route.ts`
- `app/api/exercises/categories/route.ts`
