# Workout Templates

## Segment goal

Templates speed up new workout creation by storing a repeatable structure.

## Main routes

- `/workout/template`
- `/workout/template/create`
- `/workout/template/[id]/edit`

## Functions

- create template
- template list
- search
- template preview
- edit
- delete
- use template as the base for a new workout

## Backend and API

- `app/api/workout-templates/create/route.ts`
- `app/api/workout-templates/get/route.ts`
- `app/api/workout-templates/list/route.ts`
- `app/api/workout-templates/update/route.ts`
- `app/api/workout-templates/delete/route.ts`
